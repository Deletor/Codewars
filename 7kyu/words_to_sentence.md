## Words to sentence <br>
---
Write function which will create a string from a list of strings, separated by space.

## Example:
```
["hello", "world"] -> "hello world"
```
---
### Given code
```python
def words_to_sentence(words):
    pass
```
---
### Solution
```python
def words_to_sentence(words):
    return ' '.join(words)
```
