## Grasshopper - Function syntax debugging <br>
---
A student was working on a function and made some syntax mistakes while coding. Help them find their mistakes and fix them.

---
### Given code
```python
def main [verb, noun]
return verb + noun
```
---
### Solution
```python
def main(verb, noun): 
    return verb + noun
```
