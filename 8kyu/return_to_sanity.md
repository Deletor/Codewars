## Return to Sanity <br>
---
This function should return an object, but it's not doing what's intended. What's wrong?

---
### Given code
```python
def mystery():
    results = {
    'sanity': 'Hello'}
    return 
    results
```
---
### Solution
```python
def mystery():
    results = {
    'sanity': 'Hello'}
    return results
```
