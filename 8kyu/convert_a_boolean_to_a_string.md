## Convert a Boolean to a String <br>
---
Implement a function which convert the given boolean value into its string representation.

Note: Only valid inputs will be given.

---
### Given code
```python
def boolean_to_string(b):
    #your code here
```
---
### Solution
```python
def boolean_to_string(b):
    return str(b)
```
